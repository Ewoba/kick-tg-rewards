# Drops Crypto App

Мобильное приложение для Drops криптой на React Native + Expo.

## Установка

```bash
npm install
```

## Запуск

1. Убедитесь, что у вас установлен Expo Go на телефоне или эмулятор настроен
2. Замените `API_BASE` в `App.tsx` на ваш ngrok URL
3. Запустите:

```bash
npm start
```

## Настройка

1. После запуска ngrok, обновите `API_BASE` в `App.tsx` на ваш публичный URL
2. Убедитесь, что схема диплинка `dropscrypto` совпадает с настройками в backend
